Unisciti alla famiglia di sviluppatori
Entra nella comunità di sviluppatori Unifacile. Unifacile è nato da un piccolissimo gruppo di studenti con una forta passione per l'informatica, adesso quello che vogliamo fare è creare all'interno dell'università una comunità di studenti sviluppatori che possano crescere insieme e lavorare su progetti condivisi.

Utilizza le API di Unifacile e dai sfogo alla tua immaginazione

Ancora oggi, per vari motivi, l'università è blindata dietro ai suoi siti web vecchio stampo o ad aree gestionali non sempre userfriendly.

Con le API di Unifacile potrai interagire in via sperimentale con queste informazioni e creare per i migliaia di studenti che usano Unifacile tutti giorni nuovi strumenti e funzionalit&garave; che fin ora non sono mai stati pensati.

Technologies we
Inutile dire che siamo amanti di tutte le tecnologie esistenti, queste che vi elenchiamo di seguito sono quelle che noi usiamo più spesso che potrebbero rappresentare per un voi una via d'accesso a diventare uno sviluppatore della piattaforma stessa.

PHP

La nostra piattaforma nasce è scritta in PHP. Usiamo Symfony 2, un framework php tra i più famosi in assoluto. Se Zend vi ha stancato o volete iniziare ad imparare un framework php, questa è la scelta.

JavaScript

Per rendere l'esperienza utente più facile ed accattivante utilizziamo molto JavaScript. AngularJS è il nostro framework di riferimento ma non mancano richiami a tantissime altre librerie.

MySQL

Tutti i dati che potete trovare sul sito sono salvati in un database MySQL.
